### Directory structure

- **`app/`**: Core logic and UI.
    - **`assets/`**: Static files.
        - **`css/main.css`**: Tailwind 4 theme & variables.
    - **`components/`**: Reusable components.
        - **`charts/`**: Chart.js components.
        - **`elements/`**: General UI elements.
    - **`pages/`**: App routes (pages).
    - **`stores/`**: Pinia state.
- **`public/`**: Public static assets.
- **`plugins/`**: Plugins.

### Getting Started

1. **Install**:
   ```bash
   npm install
   ```

2. **Run**:
   ```bash
   npm run dev
   ```

3. **Build**:
   ```bash
   npm run build
   ```
